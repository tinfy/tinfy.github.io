Markdown语法

发表于 2021-03-26 更新于 2021-03-27 分类于 博客搭建 阅读次数： 本文字数： 999 阅读时长 ≈ 1 分钟

标题：使用#可表示 1-6 级标题，一级标题对应一个#，二级标题对应两个#，以此类推。

段落：段落后面使用一个空行来表示重新开始一个段落。

字体：单个*或_表示斜体，两个*或_表示加粗，三个*或_表示斜体加粗。

分隔线：三个以上的-或*。

删除线：在文字两端添加两个~~。

下划线：使用HTML的<u>标签。

无序列表：使用*或+或-表示无序列表。

有序列表：使用数字加上.表示有序列表。

列表嵌套：在子列表前加上四个空格。

区块引用：在段落开头使用>。

区块引用嵌套：一个>是第一层引用，两个尖括号>是第二层引用，以此类推。

区块引用中使用列表：在>后面加上无序列表或有序列表的符号即可。

列表中使用区块引用：和列表嵌套相似，在>前加上四个空格。

代码片段：用`把代码包裹。

代码块：用```把代码包裹，在```后还可以指定一种语言。

链接：一个[]里面放链接的名称，加上一个()里面放链接的href属性。

图片：一个!，加上一个[]里面放图片的alt属性，再加上一个()里面放图片的src属性或图片的base64编码。

表格：使用|来分隔不同的单元格，使用一个或以上的-来分隔表头和其他行。

  1. ---:设置内容和标题栏居右对齐。
  2. :---设置内容和标题栏居左对齐。
  3. :---:设置内容和标题栏居中对齐。
