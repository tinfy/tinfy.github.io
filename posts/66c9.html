快捷键汇总

置顶 发表于 2021-05-25 更新于 2021-11-25 阅读次数： 本文字数： 1.1k 阅读时长 ≈ 1 分钟

在日常的工作和学习中，记住一些常用的快捷键能提高我们的效率，下面列出的是一些我常用的的快捷键。

Windows 快捷键

快捷键 功能 备注
Ctrl + F 查找
Ctrl + H 替换
Ctrl + N 打开一个相同的资源管理器界面 浏览器中使用：打开一个新的窗口
Ctrl + E 选中搜索框 资源管理器/浏览器中使用
Ctrl + 鼠标滚轮 放大/缩小字体 文件/浏览器中使用
Ctrl + 0 恢复到默认字体大小 和上一个快捷键一起使用，只能在浏览器中使用
Ctrl + 鼠标拖动 快速创建一个文件/文件夹副本
Ctrl + Tab 向后切换标签页 在浏览器中使用
Ctrl + Shift + Tab 向前切换标签页 在浏览器中使用
Ctrl + Shift + Delete 清除浏览器缓存 在浏览器中使用
Ctrl + Shift + D 把所有打开的标签页都存为书签 在浏览器中使用
Ctrl + Shift + T 恢复误关的网页 在浏览器中使用
Ctrl + Shift + N 快速新建文件夹
Ctrl + Shift + Win + B 重启屏幕 出现黑屏或蓝屏时使用(Win10)
Win + S 打开搜索 Win10
Win + I 打开设置 Win10
Win + L 锁定屏幕
Win + D 快速显示桌面 再按一次可以恢复到之前的界面
Win + E 打开资源管理器
Win + P 打开投影模式 改变声音输出设备：设置>系统>声音>输出
Win + V 多重剪贴板 Win10
Win + ←或→ 左右分屏
Win + ↑或↓ 左右分屏后上下分屏 Win10
Win + Shift + S 截图 Win10
Alt + Tab 切换屏幕/标签
F2 快速重命名
F4 重复上一步动作 在Office中使用
F5 刷新
F11 打开浏览器的全屏界面 在浏览器中使用
Delete 删除光标后的第一个字符
Shift + 鼠标右击 对文件夹使用，出现在此打开Powershell窗口 在资源管理器中使用

VS Code 快捷键

快捷键 功能
Ctrl + Shift + P 显示命令面板
Ctrl + B 显示/隐藏侧边栏
Ctrl + N 新建文件
Ctrl + J 显示/隐藏控制台
Ctrl + Tab 在已经打开的多个文件之间进行切换
Win + ←或→ 在单词之间移动光标
Fn + ←或→ 将光标定位到当前行的最左侧或最右侧
Alt + ↑或↓ 将代码向上或向下移动
Alt + Shift + ↑或↓ 将代码向上或向下复制
Ctrl + Shift + K 删除整行
Ctrl + Backspace 删除光标之前的一个单词
Ctrl + delete 删除光标之后的一个单词
Ctrl + / 添加单行注释
Alt + Shift + F 代码格式化
F2 以重构的方式进行重命名
Ctrl + F 搜索
Alt + Shift 矩形区域选择

Visual Studio 快捷键

快捷键 功能
Ctrl + K，Ctrl + C 注释选定的内容
Ctrl + K，Ctrl + U 取消注释选定的内容
Ctrl + K，Ctrl + F 代码格式化
