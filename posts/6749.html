Node.js安装教程

发表于 2021-04-01 分类于 安装教程 阅读次数： 本文字数： 569 阅读时长 ≈ 1 分钟
  1. 前往Node.js官网下载.msi文件，一路点击next即可顺利安装。 安装完成后，会自动添加环境变量。windows + R再输入cmd进入命令行，输入node -v，可以打印出node.js的版本，说明安装成功。
    node.js是自带npm的，输入npm -v可以查看npm的版本。
  2. 在nodejs目录下，新建两个文件夹：node_cache,node_global，创建完后，在命令窗口中输入如下命令：
    npm config set prefix "D:\Develop\nodejs\node_global"
npm config set cache "D:\Develop\nodejs\node_cache"
  3. 进入电脑的环境变量属性，把用户变量里默认的路径改为：D:\nodejs\node_global

  5. 测试：在命令窗口中，输入如下命令，尝试安装express模块：
    npm install express -g

出现的问题：在执行npm install express -g命令时出现错误，通过下面的命令设置了镜像服务器，成功解决了该问题：

npm config set registry http://registry.npm.taobao.org
