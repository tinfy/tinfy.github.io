由于找不到 mfc110.dll， 无法继续执行代码

发表于 2021-05-05 更新于 2021-11-03 阅读次数： 本文字数： 234 阅读时长 ≈ 1 分钟

安装完 Xshell 后，启动 Xshell 报系统错误：由于找不到 mfc110.dll，无法继续执行代码。

解决方法：
前往Microsoft下载中心：https://www.microsoft.com/zh-CN/download/details.aspx?id=30679 ，下载 Visual C++ Redistributable Package 安装运行使用 Visual Studio 2012 生成的 C++ 应用程序所必需的运行时组件。

注意：64位的系统需要安装 x86 和 x64 两个组件。
