标题

使用#可表示 1-6 级标题，一级标题对应一个#，二级标题对应两个#，以此类推。

# 标题1
## 标题2
### 标题3
#### 标题4
##### 标题5
###### 标题6

段落

段落后面使用一个空行来表示重新开始一个段落。

段落一

段落二

字体

单个*或_表示斜体，两个*或_表示加粗，三个*或_表示斜体加粗。

*斜体*
**加粗**
***斜体加粗***

分隔线

三个以上的-或*。

---

删除线

在文字两端添加~~或使用HTML的<del>标签。

~~删除线~~
<del>删除线</del>

下划线

使用HTML的<u>标签。

<u>下划线</u>

无序列表

使用*或+或-表示无序列表。

* 苹果
* 香蕉
* 橘子

有序列表

使用数字加上.表示有序列表。

1. 苹果
2. 香蕉
3. 橘子

列表嵌套

在子列表前加上四个空格。

1. 水果
    * 苹果
    * 香蕉
    * 橘子
2. 蔬菜

区块引用

在段落开头使用>。

> 子曰：学而时习之，不亦说乎

区块引用嵌套

一个>是第一层引用，两个>是第二层引用，以此类推。

> 第一层引用
> > 第二层引用
> > > 第三次引用

区块引用中使用列表

在>后面加上无序列表或有序列表的符号即可。

> 区块中使用列表
> 1. 第一项
> 2. 第二项
> + 第一项
> + 第二项
> + 第三项

列表中使用区块引用

和列表嵌套相似，在>前加上四个空格。

1. 第一项
    > 区块引用
    > 区块引用
2. 第二项

代码片段

用`把代码包裹。

`new`

代码块

用```把代码包裹，在```后还可以指定一种语言对代码高亮显示。

```
for(let i = 0; i < 10; i ++) {
  console.log(i);
}
```

[]里面填链接的名称，()里面填链接的URL。

图片

![alt属性值](图片地址)

表格

使用|来分隔不同的单元格，使用一个或以上的-来分隔表头和其他行。

  1. ---:设置内容和标题栏居右对齐。
  2. :---设置内容和标题栏居左对齐。
  3. :---:设置内容和标题栏居中对齐。
|表头|表头|
|:---:|:---:|
|单元格|单元格|
|单元格|单元格|

任务清单

和有序/无序列表相似，在后面添加[ ]或[x]，[ ]表示未完成，[x]表示已完成。
注意：[x]前后要加上一个空格，表示未完成的中括号里面也有一个空格。

1. [x] HTML
2. [x] CSS
3. [ ] JavaScript

- [x] HTML
- [x] CSS
- [ ] JavaScript
